# Шрифт TT Norms Std Trial Condensed

Скопируйте в эту папку (`public/fonts`) файлы шрифта из вашего набора. Поддерживаются оба варианта названий:

**Вариант 1 (Condensed — две буквы d):**
- `TT Norms Std Trial Condensed Regular.otf`
- `TT Norms Std Trial Condensed Medium.otf`
- `TT Norms Std Trial Condensed Bold.otf`
- `TT Norms Std Trial Condensed ExtraBold.otf`

**Вариант 2 (Condense — одна d):**
- `TT Norms Std Trial Condense Regular.otf`
- `TT Norms Std Trial Condense Medium.otf`
- `TT Norms Std Trial Condense Bold.otf`
- `TT Norms Std Trial Condense ExtraBold.otf`

Шрифт подключён в `app/globals.css` и используется по всему сайту (переменная `--font-tt-norms`). Остальные начертания можно добавить при необходимости.
