# Long Stay — новостной сайт с CMS

Сайт городского медиа о Тбилиси, Батуми и Дубае с админ-панелью для управления статьями.

## Установка

```bash
npm install
```

После установки зависимостей выполните `npm run organize-assets` (если используете изображения из Figma).

## Настройка базы данных (локально)

Локально используется **SQLite** — ничего устанавливать не нужно. Файл `.env` уже с нужным `DATABASE_URL`.

Инициализируйте базу (один раз после клона):
```bash
npx prisma generate
npx prisma db push
npx prisma db seed
```

База создастся в `prisma/dev.db`. Для деплоя на Vercel используйте PostgreSQL (Neon) и смените в `prisma/schema.prisma` провайдер на `postgresql`.

## Запуск

```bash
npm run dev
```

Сайт: http://localhost:3000  
Админка: http://localhost:3000/admin  

**Данные для входа в админку (только для локальной разработки):**
- Email: admin@longstay.com
- Пароль: admin123  
В production обязательно смените пароль или создайте нового админа и удалите этого.

## Деплой

Проект готов к размещению на Vercel, Railway, Render и аналогичных платформах.

### Переменные окружения

Скопируйте `.env.example` в `.env` на сервере или задайте переменные в панели хостинга:

| Переменная              | Описание |
|-------------------------|----------|
| `DATABASE_URL`          | Строка подключения к БД (см. ниже) |
| `JWT_SECRET`            | **Обязательно в production.** Длинная случайная строка (32+ символов) для подписи JWT. Без неё приложение в production не запустится. |
| `BLOB_READ_WRITE_TOKEN` | Для загрузки картинок в статьях на Vercel. В панели Vercel: Storage → Create Database → Blob. Токен создаётся автоматически; локально можно не задавать (файлы сохраняются в `public/uploads`). |

### База данных

Используется **PostgreSQL**. Задайте `DATABASE_URL` в настройках проекта (например, из [Neon](https://neon.tech) или [Vercel Postgres](https://vercel.com/storage/postgres)). После первого деплоя один раз выполните на своей машине с прод-`DATABASE_URL` в `.env`: `npx prisma db push` и `npx prisma db seed`. Если вы уже делали деплой до того, как поля `titleEn` и `contentEn` стали необязательными, выполните в консоли БД скрипт `prisma/make-article-en-nullable.sql` или снова `npx prisma db push` с прод-URL.

### Рекомендуемые шаги при первом деплое

1. Задать в настройках проекта переменные `DATABASE_URL` и `JWT_SECRET`.
2. Собрать проект: хостинг обычно запускает `npm install` и `npm run build` (в build уже входит `prisma generate`).
3. Один раз применить схему и сид к БД: на своей машине с настроенным `DATABASE_URL` на прод-БД выполнить `npx prisma db push` и `npx prisma db seed`, либо использовать CLI/консоль хостинга, если доступна.
4. Сменить пароль администратора по умолчанию или создать нового пользователя в админке и отключить/удалить тестового.

### Примеры

- **Vercel:** подключите репозиторий, добавьте `DATABASE_URL` (Postgres) и `JWT_SECRET`, деплой — автоматический. Чтобы загрузка картинок в статьях работала, в проекте создайте Blob store (Storage → Blob) — переменная `BLOB_READ_WRITE_TOKEN` подставится сама. Миграции и сид — через локальный Prisma с прод-`DATABASE_URL` или скрипт деплоя.
- **Railway:** создайте сервис из репозитория, добавьте переменные; при необходимости создайте том для SQLite и укажите путь к файлу в `DATABASE_URL`. Команда запуска: `npm run start`.

## Структура проекта

- `/` — главная (выбор города)
- `/[city]` — статьи города (Тбилиси, Батуми, Дубай)
- `/[city]/[slug]` — страница статьи
- `/cooperation` — сотрудничество/реклама
- `/cooperation` — реклама (при скролле — «Почему работает»); `/cooperation/why-works` перенаправляет на `/cooperation#why-works`
- `/cooperation/[city]` — форматы и стоимость по городу
- `/search?q=...` — поиск по сайту
- `/about` — о проекте
- `/admin` — админ-панель (CRUD статей)

## Функционал

- **Header**: логотип, навигация по городам, переключатель языка (RU/EN), поиск, кнопка «Сотрудничество»
- **Footer**: логотип, навигация, контакт в Telegram
- **Поиск**: popup с полем ввода, результаты на отдельной странице
- **Мультиязычность**: RU/EN, сохранение в localStorage
- **Рубрика «Экскурсии»**: ведёт на внешние ссылки Tripster
- **Соцсети**: ссылки на Instagram, Telegram, TikTok по городам
- **Редактор статей**: жирный, курсив, зачёркнутый, заголовки, ссылки, изображения в тексте (TipTap)

## Шрифт TT Norms Std Trl Cnd

Сайт использует шрифт TT Norms Std Trl Cnd. Поместите файлы `.woff2` в `public/fonts/` (см. `public/fonts/README.md`). Без файлов будет использоваться системный шрифт.

## Ассеты

Картинки и иконки из Figma лежат в `public/images/`. **Перед первым запуском** выполните:

```bash
npm run organize-assets
```

Скрипт перенесёт иконки в `public/icons/` и переименует картинки в `public/images/`.
